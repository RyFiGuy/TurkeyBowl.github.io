# Turkey Bowl Countdown

A countdown website for the historic Turkey Bowl football rivalry between Loyola Blakefield and Calvert Hall in Baltimore, Maryland.

## About The Turkey Bowl

The Turkey Bowl is the longest-standing football rivalry in the country between two Catholic prep schools. The first game took place on November 12, 1920. The game is played annually on Thanksgiving Day.

## Features

- **Live Countdown Timer**: Counts down to the next Turkey Bowl on Thanksgiving Day
- **All-Time Series Record**: Displays the historical win-loss-tie record between the two schools
- **Team Information**: Current rosters and head coach information
- **Recent Results**: Results from recent Turkey Bowl games
- **Historical Context**: Information about the 100+ year rivalry
- **News & Articles**: Links to recent news coverage and articles about the game
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## Deploying to Vercel

### Option 1: Deploy via Vercel CLI

1. Install Vercel CLI:
```bash
npm install -g vercel
```

2. Navigate to the project directory:
```bash
cd turkey-bowl-countdown
```

3. Deploy:
```bash
vercel
```

### Option 2: Deploy via Vercel Dashboard

1. Go to [vercel.com](https://vercel.com)
2. Sign in or create an account
3. Click "Add New Project"
4. Import your GitHub repository (or drag and drop the folder)
5. Vercel will automatically detect the static files
6. Click "Deploy"

### Option 3: Deploy from GitHub

1. Push this code to a GitHub repository
2. Go to [vercel.com](https://vercel.com)
3. Click "Add New Project"
4. Select your GitHub repository
5. Click "Deploy"

## Local Development

To view the site locally, simply open `index.html` in your web browser. No build process is required.

Alternatively, you can use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (http-server)
npx http-server
```

Then visit `http://localhost:8000` in your browser.
